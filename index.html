Nessa página, irei exibir alguns dos projetos desenvolvidos. O primeiro, uma página de tributo, foi feita utilizando html e css em sua construção. O objetivo dela é mostrar informações sobre um dos álbums dos Beatles lançado.

No segundo projeto, foi utilizado React para construir uma lista de livros. Nessa página, é possível pesquisar livros através da barra de pesquisa que está conectada com a API do google books, os livros encontrados podem ser adicionados a lista de favoritos ou removidos, essas informações são salvas no LocalStorage do usuário.
